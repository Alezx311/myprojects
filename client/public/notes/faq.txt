Функции во временной области
RMS
rms

Описание : Среднеквадратическое значение сигнала, соответствующее его громкости. Соответствует функции «Энергия» в YAAFE, адаптированной из Musimathics Лоя [1].
Для чего он используется : получение приблизительного представления о громкости сигнала.
Диапазон : 0.0 - 1.0где 0.0не громко и 1.0очень громко.
ZCR
zcr

Описание : количество раз, когда сигнал пересекает нулевое значение в буфере.
Для чего он используется : помогает различать перкуссионные и тональные звуки. Перкуссионные звуки будут иметь случайный ZCR по буферам, где тональные звуки будут возвращать более постоянное значение.
Диапазон : 0 - ((размер буфера / 2) - 1). В Meyda размер буфера по умолчанию ( bufferSize) равен 512, поэтому диапазон ZCR по умолчанию равен 0 - 255.
Энергия
energy

Описание : Бесконечный интеграл квадрата сигнала. По словам Лати [2].
Для чего он используется : это еще один индикатор громкости сигнала.
Диапазон : 0 - bufferSize, где 0.0очень тихо и bufferSizeочень громко.


Спектральные особенности
Амплитуда Спектр
amplitudeSpectrum

Описание : это также известно как спектр звездных величин. Вычисляя быстрое преобразование Фурье (БПФ), мы получаем сигнал, представленный в частотной области. Выходные данные представляют собой массив, где каждый индекс представляет собой интервал частот (т.е. содержащий информацию о диапазоне частот), содержащий комплексное значение (действительное и мнимое). Амплитудный спектр берет этот сложный массив и вычисляет амплитуду каждого индекса. Результатом является распределение частот в сигнале вместе с их соответствующей силой. Если вы хотите узнать больше о преобразовании Фурье и различиях между анализом во временной и частотной областях, эта статья - хорошее место для начала.
Для чего он используется : очень полезен для любого рода аудиоанализа. Фактически, многие функции, извлеченные в Meyda, основаны на этом :).
Диапазон : массив половинного размера БПФ, содержащий информацию о частотах от 0 до половины частоты дискретизации. В Meyda частота дискретизации по умолчанию ( sampleRate) составляет 44100 Гц, а размер БПФ равен размеру буфера ( bufferSize) - по умолчанию 512.
Спектр мощности
powerSpectrum

Описание : Это amplitudeSpectrumквадрат.
Для чего он используется : это лучше подчеркивает различия между элементами разрешения по частоте по сравнению с амплитудным спектром.
Диапазон : массив половинного размера БПФ, содержащий информацию о частотах от 0 до половины частоты дискретизации. В Meyda частота дискретизации по умолчанию ( sampleRate) составляет 44100 Гц, а размер БПФ равен размеру буфера ( bufferSize) - по умолчанию 512.
Спектральный центроид
spectralCentroid

Описание : индикатор «яркости» данного звука, представляющий спектральный центр тяжести. Если бы вы взяли спектр, сделали из него деревянный блок и попытались сбалансировать его на пальце (поперек оси X), то центроид спектра был бы частотой, которую ваш палец «касается», когда он успешно уравновешивается.
Для чего он используется : как уже упоминалось, это количественная оценка «яркости» звука. Это можно использовать, например, для отделения бас-гитары (низкий спектральный центроид) от трубы (высокий спектральный центроид).
Диапазон : 0 - половина размера БПФ. В Meyda размер БПФ равен размеру буфера ( bufferSize) - по умолчанию 512.
Спектральная плоскостность
spectralFlatness

Описание : Плоскостность спектра. Он вычисляется с использованием соотношения между средним геометрическим и арифметическим.
Для чего он используется : определение уровня шума в звуке. Например, чистая синусоида будет иметь приближающуюся плоскостность 0.0, а белый шум - приближающуюся 1.0.
Диапазон : 0.0 - 1.0где 0.0не ровный, а 1.0очень пологий.
Спектральный поток
spectralFlux

Описание : мера того, насколько быстро меняется спектр сигнала. Он рассчитывается путем вычисления разницы между текущим спектром и спектром предыдущего кадра.
Для чего это используется : Часто соответствует «грубости» восприятия звука. Может использоваться, например, для определения тембра звука.
Диапазон : начинается в 0.0. У него нет верхнего диапазона, так как он зависит от входного сигнала.
Спектральный наклон
spectralSlope

Описание : мера того, насколько "наклонена" форма спектра. Рассчитано путем выполнения линейной регрессии по амплитудному спектру.
Для чего он используется : может использоваться для различения различных качеств голоса, таких как шипение, дыхание и обычная речь. Тесно относится к центроиду спектра и спаду спектра.
Диапазон :0.0 - 1.0
Спектральный спад
spectralRolloff

Описание : Частота, ниже которой содержится 99% энергии спектра.
Для чего он используется : может использоваться для аппроксимации максимальной частоты сигнала.
Диапазон : 0 - половина частоты дискретизации. В Meyda частота дискретизации по умолчанию ( sampleRate) составляет 44100 Гц .
Спектральное распространение
spectralSpread

Описание : указывает, насколько распространено частотное содержимое по спектру. Соответствует полосе частот.
Для чего он используется : может использоваться для различения шумных (большой спектральный разброс) и тональных звуков (низкий спектральный разброс).
Диапазон : 0 - половина размера БПФ. В Meyda размер БПФ равен размеру буфера ( bufferSize) - по умолчанию 512.
Спектральная асимметрия
spectralSkewness

Описание : указывает, смещен ли спектр в сторону определенного диапазона значений относительно его среднего значения.
Для чего он используется : часто используется, чтобы получить представление о тембре звука.
Диапазон : может быть отрицательным, положительным или 0. Где 0 симметрично относительно среднего, отрицательный указывает, что частотный контент смещен вправо от среднего, а положительный означает, что частотный контент смещен влево от среднего.
Спектральный эксцесс
spectralKurtosis

Описание : показатель остроты спектра. Можно рассматривать как противоположность Spectral Flatness.
Для чего он используется : часто используется для обозначения «высоты звука / тональности» звука.
Диапазон :, 0.0 - 1.0где 0.0не тональный, а 1.0очень тональный.
Цветность
chroma

Описание : вычисляет количество каждого класса хроматической высоты тона (C, C♯, D, D♯, E, F, F♯, G, G♯, A, A♯, B) в сигнале.
Для чего он используется : часто используется для анализа гармонического содержания записанной музыки, например для определения аккордов или тональности.
Диапазон : 0.0 - 1.0для каждого класса поля.


Перцептивные особенности
Громкость
loudness

Описание : человеческое восприятие частоты нелинейно. Шкала Барка была разработана для того, чтобы иметь шкалу, на которой равные расстояния соответствуют равным расстояниям восприятия частоты. Эта функция выводит объект с двумя значениями:
Громкость входящего звука на каждом шаге (часто называемом полосами) этой шкалы ( .specific). Всего 24 группы.
Общая громкость ( .total), которая представляет собой сумму 24 .specificкоэффициентов громкости.
Для чего он используется : может использоваться для создания фильтров, которые лучше соответствуют человеческому восприятию громкости.
Диапазон : 0.0 - 1.0для каждой .specificгромкости. 0.0 - 24.0для .totalгромкости.
Восприятие распространения
perceptualSpread

Описание : вычисляет разброс .specificкоэффициентов громкости по шкале лая.
Для чего он используется : индикатор того, насколько «насыщенным / полным» будет восприятие звука.
Диапазон : 0.0 - 1.0где 0.0не «богатый», а 1.0очень «богатый».
Перцептивная резкость
perceptualSharpness

Описание : Воспринимаемая «резкость» звука на основе коэффициентов громкости Барка.
Для чего он используется : определение того, воспринимается ли входящий звук как «резкий». Может использоваться, например, для различения звуков малого и большого барабана.
Диапазон : 0.0 - 1.0где 0.0не «острый» (например, 1.0большой барабан), а очень острый (например, малый барабан).
Кепстральные коэффициенты Mel-частоты
mfcc

Описание : поскольку люди не интерпретируют высоту звука линейно, были разработаны различные шкалы частот, чтобы представить, как люди слышат расстояние между тонами. Шкала Mel - одна из них, и теперь она широко используется для приложений, связанных с голосовой связью. Реализация Meyda была вдохновлена ​​Хуангом [3], Дэвисом [4], Грирсоном [5] и библиотекой librosa .
Для чего он используется : часто используется для обнаружения голосовой активности (VAD) перед автоматическим распознаванием речи (ASR).
Диапазон : массив значений, представляющих интенсивность для каждой полосы Mel. По умолчанию размер массива равен 13, но его можно настроить с помощью numberOfMFCCCoefficients.


Коммунальные экстракторы
Комплексный спектр
complexSpectrum

Описание : массив комплексных значений ( ComplexArray), содержащий как действительную, так и мнимую части БПФ.
Для чего он используется : для создания amplitudeSpectrum. Он также используется для дальнейшей обработки сигнала, поскольку содержит информацию как о частоте, так и о фазе сигнала.
Диапазон : массив половинного размера БПФ, содержащий информацию о частотах от 0 до половины частоты дискретизации и соответствующих им значениях фазы. В Meyda частота дискретизации по умолчанию ( sampleRate) составляет 44100 Гц, а размер БПФ равен размеру буфера ( bufferSize) - по умолчанию 512.
Буфер
buffer

Описание : это необработанный звук, который вы получаете при чтении входного сигнала с микрофона, файла WAV или любого другого входного аудио. Он кодируется как Float32Array.
Для чего он используется : все функции временной области в Meyda извлекаются из этого буфера. Вы также можете использовать это для визуализации звуковой волны.
Диапазон : массив размера bufferSize, в котором каждое значение может находиться в диапазоне -1.0 - 1.0.


Оконные функции
Функции оконного управления используются во время преобразования сигнала из временной области (т. Е. Давления воздуха во времени) в частотную область (фаза и интенсивность каждой синусоидальной волны, составляющей сигнал); необходимое условие для многих описанных выше звуковых функций. Оконные функции генерируют огибающую чисел от 0 до 1 и умножают эти числа точечно на каждую выборку в сигнальном буфере, делая выборки в середине буфера относительно громче и делая выборки на обоих концах буфера относительно тише. Это сглаживает результат преобразования в частотную область, что делает окончательные звуковые характеристики более согласованными и менее дрожащими.

Meyda поддерживает 4 оконных функции, каждая из которых имеет разные характеристики. Дополнительные сведения об использовании окон см. В этой статье . По умолчанию Meyda применяет к сигналам окно хэннинга, а не прямоугольное окно, перед их преобразованием в частотную область.

Ханнинг
Meyda.windowing(signalToBeWindowed, "hanning");

Хэмминга
Meyda.windowing(signalToBeWindowed, "hamming");

Blackman
Meyda.windowing(signalToBeWindowed, "blackman");

Синус
Meyda.windowing(signalToBeWindowed, "sine");

Прямоугольный (без окна)
Meyda.windowing(signalToBeWindowed, "rect");
